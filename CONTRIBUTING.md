# Contributing to NggaPrinter

Terima kasih telah tertarik untuk berkontribusi pada **NggaPrinter**! Kami sangat menghargai bantuan Anda untuk menjadikan library ini lebih baik.

## 🛠️ Cara Berkontribusi

### Melaporkan Bug
- Pastikan bug tersebut belum dilaporkan sebelumnya di bagian [Issues](https://github.com/ringga-dev/Printer-ESC-POS/issues).
- Jika belum ada, buat issue baru dengan deskripsi yang jelas, langkah-langkah untuk mereproduksi bug, dan ekspektasi hasilnya.

### Menyarankan Fitur Baru
- Kami selalu terbuka untuk ide baru! Silakan buka issue baru dan gunakan label `enhancement`.

### Mengirimkan Pull Request (PR)
1. Fork repository ini.
2. Buat branch baru untuk fitur atau perbaikan Anda (`git checkout -b fitur-baru`).
3. Pastikan kode Anda mengikuti gaya penulisan Kotlin yang standar.
4. Pastikan build berhasil dijalankan dengan `./gradlew build`.
5. Kirim Pull Request ke branch `master`.

## 📜 Kode Etik
Harap menjaga profesionalisme dan saling menghargai dalam setiap interaksi di komunitas ini.
