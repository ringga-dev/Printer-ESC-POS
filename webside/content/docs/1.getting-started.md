## Introduction

**KmpPrinter** is a powerful, flexible, and easy-to-use Kotlin Multiplatform library for ESC/POS thermal printing. It supports a wide range of platforms and connection types, making it the ideal choice for modern cross-platform applications.

## Installation

Add the dependency to your `commonMain` source set in your `build.gradle.kts` file:

```kotlin
kotlin {
    sourceSets {
        val commonMain by getting {
            dependencies {
                implementation("ngga.ring:printer:1.0.0")
            }
        }
    }
}
```

## Basic Usage

The main entry point to the library is the `KmpPrinter` class. It acts as a coordinator for discovery, connection, and printing operations.

### Initialization

Create an instance of `KmpPrinter` in your application:

```kotlin
val printer = KmpPrinter()
```

### Checking Permissions

Before searching for or connecting to a printer, you may need to request specific permissions (especially on Android).

```kotlin
printer.checkAndRequestPermissions("BLUETOOTH") { granted ->
    if (granted) {
        // Proceed with discovery or printing
    } else {
        // Handle permission denied
    }
}
```

## Supported Platforms

- **Android**: Bluetooth (Classic/BLE), USB, Network.
- **iOS**: Bluetooth (BLE), Network.
- **JVM/Desktop**: USB, Network, Bluetooth (Platform dependent).
- **Web (Wasm/JS)**: WebUSB, Network.
