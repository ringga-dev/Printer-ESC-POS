## Discovery

KmpPrinter provides a unified discovery API to find nearby printers.

```kotlin
printer.discovery("BLUETOOTH").collect { printers ->
    printers.forEach { device ->
        println("Found: ${device.name} at ${device.address}")
    }
}
```

### Connection Types

You can specify the connection type as a string:
- `"BLUETOOTH"`
- `"USB"`
- `"NETWORK"`

## Connecting to a Printer

To connect, you first need a `PrinterConfig` which contains the address and connection parameters.

```kotlin
val config = PrinterConfig(
    address = "00:11:22:33:44:55",
    connectionType = "BLUETOOTH",
    name = "Thermal Printer"
)
```

## Connection State

You can observe the connection state in real-time:

```kotlin
printer.connectionState.collect { state ->
    when (state) {
        is ConnectionState.Connected -> println("Connected to ${state.name}")
        is ConnectionState.Connecting -> println("Connecting...")
        is ConnectionState.Disconnected -> println("Disconnected")
        is ConnectionState.Error -> println("Error: ${state.message}")
    }
}
```

## Manual Disconnection

When you're done printing, it's good practice to disconnect:

```kotlin
printer.disconnect()
```
