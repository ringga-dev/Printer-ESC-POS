## The DSL Builder

KmpPrinter uses an expressive Kotlin DSL to build raw ESC/POS streams. You don't need to know hex codes; just use the human-readable methods.

```kotlin
printer.print(config) {
    initialize()
    alignCenter()
    bigFont()
    line("MY STORE")
    normalFont()
    line("Jalan Merdeka No. 1")
    divider()
    
    // Left-Right Text (Segmented Line)
    segmentedLine("Invoice ID", "#12345")
    segmentedLine("Date", "2026-05-08")
    
    breakLine()
    
    // Table with Weights
    tableRow(
        columns = listOf("Item", "Qty", "Price"),
        weights = listOf(2, 1, 1) // 50% / 25% / 25%
    )
    divider('-')
    
    tableRow(listOf("Latte", "1", "$4.50"), listOf(2, 1, 1))
    tableRow(listOf("Croissant", "2", "$6.00"), listOf(2, 1, 1))
    
    divider()
    alignRight()
    bold(true)
    line("TOTAL: $10.50")
    bold(false)
    
    feed(3)
    cut()
}
```

## Text Alignment

You can set alignment globally for subsequent lines or use helper methods:

- `alignLeft()`: Default alignment.
- `alignCenter()`: Centers all following text/images.
- `alignRight()`: Right-aligns all following text/images.
- `centerText(text)`: Centers a single line of text.

## Left-Right Text (Segmented)

The `segmentedLine(left, right)` method is specifically designed for receipt key-value pairs. It automatically calculates the necessary spaces to push the `right` string to the far right edge of the paper.

```kotlin
segmentedLine("Total Item", "12")
segmentedLine("Payment Method", "OVO")
```

## Tables

For more than two columns, use `tableRow`. It takes a list of strings and a list of weights to determine column widths.

```kotlin
tableRow(
    columns = listOf("Product", "Qty", "Subtotal"),
    weights = listOf(3, 1, 2)
)
```

## Image Printing

Use `imageAutoScale` for the easiest integration. It automatically resizes your image to fit the printer's paper width perfectly.

```kotlin
imageAutoScale(
    grayscale = imageArray,
    width = 1024,
    height = 768,
    dithering = "FLOYD_STEINBERG" // High quality
)
```
