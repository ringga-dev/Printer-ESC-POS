## Real-time Monitoring

Many ESC/POS printers support status queries. KmpPrinter provides a simple way to monitor these statuses in the background.

```kotlin
printer.monitorStatus(config, intervalMs = 2000).collect { status ->
    if (!status.isOnline) {
        println("Printer is offline!")
    }
    if (status.isPaperOut) {
        println("Please replace the paper roll.")
    }
    if (status.isCoverOpen) {
        println("Printer cover is open.")
    }
}
```

## One-time Query

You can also query the status manually at any time:

```kotlin
val status = printer.queryStatus()
println("Is Online: ${status.isOnline}")
```

## Important Note

Status monitoring requires a bi-directional connection. Some Bluetooth Low Energy (BLE) printers or specific USB adapters might not support status queries correctly depending on the implementation.
